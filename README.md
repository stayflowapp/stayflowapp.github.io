# StayFlow V2 Fixed

A static PWA build for GitHub Pages.

## Upload
Upload the **contents** of this folder to the root of your GitHub repository.

## Activation Code
`STAYFLOW-4921`

## Notes
- No React or Tailwind build step required.
- Uses localStorage for offline-first data.
- Setup screen and dashboard are properly separated.
- Includes Rooms, Bookings, Inventory, Settings, export/import backup, dark mode, and installable PWA support.
