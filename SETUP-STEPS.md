# Setup steps

## Property type toggle recommendation
Yes — the property should declare its type during setup:
- Guest House
- Lodge
- Apartment

This should not create separate complicated apps. Instead, it should:
- preload relevant issue templates
- slightly adapt labels and examples
- keep one consistent dashboard structure

That is the best balance between simplicity and usefulness.

## Why not fully separate dashboards?
A single dashboard framework is easier to maintain and easier for users to learn.
The property type should shape defaults, not split the product into three different applications.
